## 1. 产品概述

共享琴房练琴 Web 应用，解决音乐学员预约琴房练琴的场景需求。核心能力包括：琴房资源管理、时段预约与合并拆分、家庭账户共享练琴额度、多人并发预约时的额度防超扣与实时余额一致性。

- 目标用户：琴房运营方、学琴学员（含家庭账户成员）
- 产品价值：提升琴房利用率，简化预约流程，保障额度共享时的数据一致性

## 2. 核心功能

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|

5. **练琴统计**：时长统计图表、成员使用排行

## 3. 核心流程

学员登录后进入首页，查看家庭共享额度余额。点击琴房排期进入周视图，选择目标琴房和连续时段（支持多时段合并）。提交预约时系统校验额度是否充足，并通过加锁机制防止并发超额。预约成功后若相邻时段同属该学员则自动合并为一条占用记录。退订时若退订的是合并占用的中间部分，则自动拆分为前后两段独立预约。

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深墨绿 `#1B4332`（呼应钢琴与优雅感），辅以暖金色 `#D4A373` 点缀
- **次色调**：米白 `#FAF8F5` 背景、石板灰 `#4A5568` 文字
- **按钮风格**：圆角 8px，主要按钮使用深墨绿填充配白色文字，悬停时加轻微阴影与上浮动效
- **字体**：标题使用 `Noto Serif SC`（衬线体，体现音乐艺术感），正文使用 `Noto Sans SC`
- **布局风格**：卡片式布局，顶部固定导航栏，内容区采用栅格系统
- **图标风格**：使用 Lucide 线性图标，与深墨绿主题色统一

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 首页 | 额度卡片 | 大号数字展示剩余额度，带渐变背景与微发光效果 |
| 首页 | 琴房快览 | 横向滚动卡片，显示琴房名称、类型、当前状态 |
| 琴房排期 | 周视图排期表 | 表头横向日期 + 琴房纵向排列，30 分钟粒度色块，已占用灰色、可选悬停高亮金色 |
| 琴房排期 | 预约确认 | 底部抽屉式面板，显示时段汇总、额度计算、确认按钮 |
| 我的预约 | 预约列表 | 时间轴布局，合并预约显示为加粗色块，退订按钮悬停变红 |

### 4.3 响应式

- 桌面端优先设计，主内容区最大宽度 1280px
- 排期表在平板端横向滚动，移动端切换为纵向单琴房视图
- 所有触摸目标 ≥ 44px，移动端导航栏底部固定

## 5. 核心业务规则

### 5.1 占用合并规则
- 同一学员、同一琴房、日期连续且时段首尾相接（无间隔）→ 自动合并为一条占用记录
- 合并后记录的开始时间为最早时段，结束时间为最晚时段
- 合并记录在界面上显示为连续色块，不可单独退订其中部分（需拆分后退订）

### 5.2 占用拆分规则
- 退订合并占用中的非起始/非结束时段 → 自动拆分为前后两条独立占用
- 拆分后各自独立计费，退回对应时段的额度
- 退订起始时段 → 原记录缩短；退订结束时段 → 原记录缩短；退订全部 → 删除记录并全额退款

### 5.3 共享额度与并发扣减
- 家庭账户共享一个额度池，所有成员共用
- 提交预约时使用分布式锁（基于数据库乐观锁 + 版本号）防止并发超额
- 扣减公式：`所需额度 = 预约总分钟数 / 60`（按小时计费，向上取整到 0.5 小时）
- 额度不足时拒绝预约，前端实时展示余额变动
